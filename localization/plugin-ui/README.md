# Каталог переводов окон плагинов

Этот каталог предназначен для будущего перевода окон плагинов через Weblate или
похожий инструмент.

Пока runtime-сборка использует Win32 `.rc`-файлы самих плагинов. Следующий шаг —
вынести переводимые подписи из `.rc` в стабильный текстовый каталог и
генерировать из него resource-фрагменты для сборки.

Планируемая структура ключей:

```text
export_docx.dialog.settings.caption
export_docx.option.export_images
export_epub.dialog.settings.caption
import_epub.error.stage.open_file
export_html.option.include_document_description
```

Требования к ключам:

- ключ не должен меняться при переформулировке текста;
- в ключе не должно быть координат, размеров и Win32-style флагов;
- один пользовательский текст — один ключ;
- если строка содержит имя формата (`FB2`, `EPUB`, `DOCX`), его лучше оставить
  неизменным во всех языках.

Целевые языки первого этапа:

- `en-US`;
- `ru-RU`;
- `uk-UA`;
- `de-DE`;
- `fr-FR`;
- `es-ES`;
- `it-IT`;
- `pl-PL`;
- `pt-PT`;
- `nl-NL`;
- `cs-CZ`;
- `bg-BG`.
