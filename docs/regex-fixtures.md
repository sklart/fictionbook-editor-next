# Набор regex-кейсов

Файл [regex-fixtures.json](D:\Download\FBeditor\tools\tests\regex-fixtures.json)
содержит канонические regex-кейсы для автоматических и ручных проверок FBE.

## Зачем он нужен

- хранить реальные кейсы, уже встречавшиеся в FBE;
- не тащить в тесты случайные внешние наборы из интернета;
- отдельно фиксировать ожидания для разных движков:
  - `Scintilla` / `Lexilla` в режиме `Код`;
  - `PCRE2` в режиме `Дизайн`;
  - legacy `PCRE 7.9` в compare-тестах миграции, если доступен внешний `pcre.dll`.

## Правила пополнения

Добавлять в набор:

- кейсы из реальных пользовательских багов;
- кейсы после обновления `Scintilla`, `Lexilla`, `PCRE2`, `Hunspell`, если они
  затрагивают regex-поведение;
- минимальные примеры, которые можно быстро проверить вручную и автоматически.

Не стоит добавлять:

- большие случайные корпуса regex из интернета без адаптации;
- кейсы без явного ожидаемого результата;
- дубли, которые не добавляют нового поведения.

## Минимальные обязательные сценарии

- круглые скобки как группы захвата;
- круглые скобки как литералы;
- backreference в замене;
- повторный поиск по нескольким совпадениям;
- UTF-8 и кириллица;
- multiline;
- caseless.
