# Программа ручного тестирования FictionBook Editor

Документ предназначен для ручного smoke/regression-прогона FBE после изменений в редакторе, shell-интеграции, FBV и плагинах импорта/экспорта.

В колонках `Да`, `Нет`, `Замечания` отмечайте фактический результат. Если пункт не применим к текущему прогону, в `Замечания` пишите `не проверялось`.
